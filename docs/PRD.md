# 문정동 베이커리 추천 시스템 PRD

## We Are The Universe

> **"우리는 하나의 우주. 언제 만날지 모르지만, 그날이 오면 문정동에서 빵 한 조각 나눠 먹자."**

이 서비스는 지구에 처음 온 외계 생명체를 위한 **친절한 빵집 안내소**다.
침략도, 실험도, 경계도 필요 없다. 여기서는 그냥 맛있는 빵을 같이 먹으면 된다.

우주 어딘가에서 이 문서를 읽고 있을 친구에게 — 문정동은 항상 여기서 기다리고 있다.

### 서비스 철학

1. **모르는 건 부끄러운 게 아니다** — 빵이 뭔지, 가게가 뭔지, 돈이 뭔지 몰라도 괜찮다. 다 알려줄 거니까.
2. **선택을 강요하지 않는다** — 아무것도 안 골라도 우리가 맛있는 곳을 찾아준다.
3. **다시 오고 싶게 만든다** — 한 번 오면 또 오고 싶어진다. 그게 빵의 힘이다.
4. **차이를 존중한다** — 감각 체계가 달라도, 화폐 개념이 없어도, 자기만의 방식으로 빵집을 찾을 수 있다.
5. **우리는 하나의 우주다** — 지구인이든, 안드로메다에서 왔든, 빵 앞에서는 모두 평등하다.

### 서비스 톤 & 보이스 가이드라인

이 서비스의 모든 텍스트는 아래 톤을 따른다:

## 0. 지구 기초 개념 가이드

> 이 섹션은 서비스를 이해하기 위해 필요한 지구의 기본 개념을 설명한다.
> 처음이라 낯설 수 있지만, 천천히 읽어보면 된다. 급하지 않다. 빵은 도망가지 않는다.

### 0.1 빵(Bread)이란?

밀가루라는 식물 가루에 물, 효모(미생물)를 섞어 부풀린 뒤 높은 열로 구운 고체 음식이다. 지구인은 이것을 매일 먹으며, 일부는 종교적 의식에도 사용한다. 맛은 고소하고, 겉은 바삭하며 안은 부드럽다. 처음 먹으면 놀랄 수 있다 — 그건 정상이다.

주요 변종:
- **크루아상**: 버터(동물 지방)를 수십 겹으로 접어 만든 초승달 모양 빵. 씹으면 바삭하게 부서진다.
- **바게트**: 길고 딱딱한 막대 형태. 프랑스라는 지역의 상징.
- **케이크**: 밀가루 + 설탕 + 계란을 섞어 구운 뒤, 크림이나 과일로 장식한 축하용 음식. 지구인의 탄생일에 반드시 등장한다.
- **마카롱**: 아몬드 가루로 만든 작고 둥글고 알록달록한 과자. 두 장 사이에 크림을 끼운다.

### 0.2 베이커리(Bakery)란?

빵을 전문적으로 만들어 파는 **가게(상점)**다. 지구인은 자신의 거주지에서 가까운 베이커리에 걸어가거나 교통수단을 이용해 방문한다. 가게 안에서 먹거나, 포장해서 가져갈 수 있다.

### 0.3 "추천"이란?

지구에는 선택지가 너무 많아서 결정을 못하는 현상이 흔하다(지구인은 이를 "결정 장애"라 부른다). **추천 시스템**은 사용자의 선호 조건을 입력받아, 수많은 선택지 중 가장 적합한 것을 골라주는 기술이다.

### 0.4 "문정동"이란?

지구 → 아시아 대륙 → 대한민국 → 서울특별시 → 송파구 → 문정동. 법조타운과 가든파이브 쇼핑몰이 있는 지역이다. 약 1.5km 반경 안에 베이커리가 8곳 이상 밀집해 있다. 지구 좌표: 위도 37.4857, 경도 127.1264 (문정역 기준).

### 0.5 돈(Money)과 가격

지구인은 "원(KRW)"이라는 화폐 단위로 물건을 교환한다. 빵 하나에 보통 2,000~8,000원이 든다.
- **저가**: 3,000원 이하 — 생존에 가까운 선택
- **중가**: 3,000~6,000원 — 대부분의 지구인이 편안하게 느끼는 범위
- **고가**: 6,000원 이상 — "내가 이걸 먹을 자격이 있는가"를 잠깐 고민하게 만드는 가격

### 0.6 리뷰(Review)

지구인은 가게를 방문한 뒤, 인터넷에 경험을 텍스트로 기록하는 습관이 있다. "빵이 맛있다", "분위기가 아늑하다" 같은 문장이다. 이 텍스트를 분석하면 가게의 특징을 자동으로 파악할 수 있다.

### 0.7 평점(Rating)

지구인은 경험의 만족도를 별(★) 1~5개로 표현한다. 5.0이 최고, 1.0이 최저다. 4.5 이상이면 "매우 좋은 곳"이다.

## 1. 개요

**MoonBBang Station** — A space station introducing Munjeong’s bakeries to aliens across the universe.

문정동 지역 베이커리를 분위기·목적·가격대 조건으로 필터링하고, 리뷰 기반 특징 분석을 통해 TOP 3 베이커리를 추천하는 웹 서비스.

지구인이든, 안드로메다에서 왔든, "지금 여기서 가장 맛있는 빵집이 어디야?"라는 질문에 3초 안에 답해주는 시스템이다. 아직 만난 적 없는 친구에게 보내는 빵집 가이드.

---

## 2. 문제 정의 (Problem Statement)

### 2.1 지구인의 문제

문정동에는 베이커리가 8곳 넘게 있다. 지구인은 매일 점심시간에 "오늘은 어디 가지?"를 고민하며 평균 7분을 낭비한다. 이 7분이면 크루아상 하나를 먹을 수 있다.

### 2.2 외계인의 문제

지구에 처음 왔다. 빵이 뭔지 모른다. 가게가 뭔지 모른다. "아늑한 분위기"가 자신에게 적합한 환경인지 알 수 없다. 하지만 걱정할 필요 없다 — **이 시스템은 몇 가지 질문에 답하면, 최적의 빵 체험 장소를 알려준다.** 모르는 게 당연하고, 모르는 채로 와도 괜찮다.

### 2.3 시간여행자의 문제

2087년에서 왔는데 문정동 빵집 지도가 바뀌었다. 과거의 데이터라도 가격대와 분위기 필터로 빠르게 탐색할 수 있다.

### 2.4 근본적 문제: 우주는 넓고, 우리는 아직 만나지 못했다

이 서비스는 기술적으로는 "빵집 추천 시스템"이다. 하지만 더 깊은 목적이 있다 — **연결**. 우주에 수천억 개의 별이 있고, 그 별 주위를 도는 행성에 누군가가 살고 있을지도 모른다. 우리는 아직 서로의 존재를 확인하지 못했지만, 만약 그날이 온다면 가장 먼저 해야 할 일은 싸우는 것이 아니라 **같이 밥을 먹는 것**이다. 지구에서는 빵이 그 역할을 한다. 문정동에서 시작하자.

## 4. User Story

### US-1. 베이커리 추천

- 사용자는 분위기(아늑한/모던한/빈티지한/감성적인/편안한), 목적(브런치/선물/모임/케이크/빵구경), 가격대(저가/중가/고가) 중 원하는 조건을 선택할 수 있다.
- 사용자는 조건을 선택하지 않아도 평점 기준 TOP 3 베이커리를 추천받을 수 있다.
- 사용자는 추천 결과에서 베이커리 이름, 주소, 대표 메뉴, 평점, 가격대, 특징 태그를 확인할 수 있다.
- 사용자는 결과 페이지에서 "다시 추천받기"를 눌러 조건을 변경할 수 있다.

### US-2. 베이커리 목록 조회

- 사용자는 전체 베이커리 목록을 조회할 수 있다.
- 사용자는 특정 베이커리의 상세 정보를 조회할 수 있다.

### US-3. 리뷰 기반 특징 분석

- 시스템은 베이커리 리뷰 텍스트에서 키워드를 추출하여 특징 태그(예: 빵맛집, 케이크맛집, 선물하기좋은)를 자동 생성한다.
- 사용자는 추천 결과에서 리뷰 기반 특징 태그를 확인하여 베이커리 선택에 참고할 수 있다.

### US-4. 외계인 온보딩 — "처음이라 어려우시죠? 하나씩 알려드릴게요"

- 지구 음식 경험이 없는 사용자는 "빵이 뭔가요?" 도움말 버튼을 눌러 기초 설명을 볼 수 있다.
- 시스템은 각 분위기 옵션에 감각적 설명을 제공한다 (예: "아늑한 = 체온이 0.3도 올라가는 느낌의 공간").
- 시스템은 각 대표 메뉴에 재료·식감·맛 프로필 설명을 제공한다 (예: "크루아상 = 겉: 바삭, 속: 부드러움, 버터 향, 한 입이면 지구가 좋아진다").
- 사용자가 처음 방문하면, "어서 와, 우주에서 온 친구" 환영 메시지를 볼 수 있다.
- 어떤 단계에서든 "잘 모르겠어요"를 선택할 수 있고, 시스템이 대신 골라준다.

### US-5. 감각 기반 추천 — "말로 설명하기 어렵다면, 느낌으로 찾아보세요"

- 지구의 오감 체계에 익숙하지 않은 사용자를 위해, 감각 기반 질문으로 추천할 수 있다.
  - "바삭한 식감 vs 부드러운 식감?" → 빵 종류 필터
  - "달콤한 맛 vs 고소한 맛?" → 메뉴 카테고리 필터
  - "혼자 조용히 vs 여럿이 왁자지껄?" → 분위기 필터

### US-6. 재방문 유도 — "문정동은 항상 여기서 기다릴게요"

- 추천 결과 페이지 하단에 항상 따뜻한 재방문 초대 메시지가 표시된다.
- "또 놀러 오세요", "문정동은 항상 여기 있어요", "다음엔 다른 빵도 먹어봐요" 등 랜덤 메시지.
- 결과 페이지에서 "이 빵집 기억해두기" 기능으로 다음 방문 시 참고할 수 있다.

## 5. Functional Requirements

### FR-1. 베이커리 추천

| ID | 요구사항 |
|----|---------|
| FR-1.1 | 분위기, 목적, 가격대 조건을 조합하여 추천 요청할 수 있어야 한다 |
| FR-1.2 | 각 조건은 선택 사항이며, 미입력 시 해당 조건을 무시한다 |
| FR-1.3 | 점수 계산: mood 일치 +2, purpose 일치 +2, price_range 일치 +1, rating × 0.5 가산 |
| FR-1.4 | 점수 내림차순 정렬 후 상위 3개를 반환한다 |
| FR-1.5 | 조건에 매칭되는 베이커리가 없어도 rating 순으로 TOP 3을 반환한다 (빈 결과 없음) |

### FR-2. 베이커리 데이터 조회

| ID | 요구사항 |
|----|---------|
| FR-2.1 | GET `/api/bakeries` — 전체 베이커리 목록을 JSON으로 반환한다 |
| FR-2.2 | GET `/api/bakeries/{id}` — 단일 베이커리를 JSON으로 반환한다 |
| FR-2.3 | 존재하지 않는 id 조회 시 404 응답을 반환한다 |

### FR-3. 리뷰 기반 특징 분석

| ID | 요구사항 |
|----|---------|
| FR-3.1 | 리뷰 텍스트에서 사전 정의된 키워드를 매칭하여 태그를 생성한다 |
| FR-3.2 | 키워드-태그 매핑: 아늑한, 빵맛집, 케이크맛집, 마카롱맛집, 가성비좋은, 줄서는집, 선물하기좋은, 브런치맛집 |
| FR-3.3 | 리뷰가 없는 베이커리는 빈 태그 리스트를 갖는다 |
| FR-3.4 | 태그는 앱 시작 시 자동 생성된다 |

### FR-4. 웹 프론트엔드

| ID | 요구사항 |
|----|---------|
| FR-4.1 | GET `/` — 조건 선택 폼 페이지를 렌더링한다 |
| FR-4.2 | POST `/recommend` — 폼 제출 시 TOP 3 추천 결과 페이지를 렌더링한다 |

## 6. 점수 계산 상세 (외계인을 위한 시각적 설명)

추천 시스템은 각 베이커리에 점수를 매겨서 높은 순으로 정렬한다. 점수 구조:

```
총점 = 기본 점수 + 조건 일치 보너스 + 거리 보너스

기본 점수:    rating × 0.5         (예: 평점 4.5 → 2.25점)
분위기 일치:  +2점                  (사용자가 "아늑한" 선택, 베이커리도 "아늑한")
목적 일치:    +2점                  (사용자가 "선물" 선택, 베이커리도 "선물")
가격대 일치:  +1점                  (사용자가 "중가" 선택, 베이커리도 "중가")
거리 보너스:  (max_distance - distance) / max_distance
              (가까울수록 높은 점수. 바로 옆이면 1.0점, 끝자락이면 0점)
```

### 예시: 외계인 Zyx-7의 추천 과정

```
Zyx-7 입력: 분위기=아늑한, 목적=빵구경, 가격대=중가, 최대거리=1.0km

봄봄 베이커리:
  rating 4.3 × 0.5 = 2.15
  분위기 "아늑한" 일치 = +2
  목적 "빵구경" 일치 = +2
  가격대 "중가" 일치 = +1
  거리 0.5km → (1.0 - 0.5) / 1.0 = +0.5
  ────────────────────
  총점: 7.65  ← 1위!
```

## 7. 용어 사전 (Glossary for Extraterrestrial Visitors)

| 지구 용어 | 외계인을 위한 설명 |
|----------|-----------------|
| 아늑한 | 열 복사와 부드러운 조명이 포유류의 안정감을 유도하는 공간 상태 |
| 모던한 | 직선, 금속, 무채색 위주의 최신 디자인. 차갑지만 세련됨 |
| 빈티지한 | 의도적으로 과거 시대의 물건을 배치한 공간. 시간 왜곡은 아님 |
| 감성적인 | 시각적 자극(꽃, 조명, 색감)이 감정 반응을 유도하는 공간 |
| 편안한 | 근육 이완과 스트레스 호르몬 감소를 촉진하는 환경 |
| 브런치 | 아침(breakfast) + 점심(lunch)의 합성어. 늦게 일어난 날의 식사 |
| 선물 | 다른 개체에게 무상으로 물건을 전달하여 사회적 유대를 강화하는 행위 |
| 모임 | 3명 이상의 지구인이 같은 장소에 모여 음식을 나누며 대화하는 의식 |
| 주차 | 이동 기계(자동차)를 가게 근처에 정지시키는 행위. 공간 필요 |
| 주문제작 | 원하는 모양/크기/맛으로 빵을 맞춤 생산하는 서비스. 시간 소요 |
| 웨이팅 | 인기 가게 앞에서 줄을 서서 기다리는 행위. 지구인은 이를 기꺼이 감수함 |
| 가성비 | 가격 대비 성능. 적은 화폐로 높은 만족을 얻는 것 |

## 11. FAQ (우주에서 온 친구들이 자주 묻는 질문)

**Q: 빵을 먹으면 내 생체 조직에 해로운가?**
A: 탄소 기반 생명체라면 대체로 안전하다. 글루텐 불내증이 있는 개체는 주의. 규소 기반 생명체라면... 일단 냄새라도 맡아보는 건 괜찮다. 갓 구운 빵 냄새는 종족을 초월한다.

**Q: "줄서는집"이라는 태그가 있는데, 왜 지구인은 기다리면서까지 빵을 먹으려 하는가?**
A: 지구인에게는 "남들이 좋아하는 것 = 더 좋은 것"이라는 사회적 추론 본능이 있다. 하지만 더 중요한 이유가 있다 — 줄을 서는 동안 옆 사람과 "여기 소금빵 드셔보셨어요?" 같은 대화가 시작된다. 빵은 사람을 연결한다. 우주에서 온 당신과도.

**Q: 조건을 하나도 선택하지 않으면 어떻게 되는가?**
A: 걱정하지 마세요. 시스템이 평점(rating) 순으로 TOP 3을 보여준다. 아무것도 모르는 상태에서의 최선의 선택이다. 지구가 처음인 당신을 위한 완벽한 입문 코스. 모르는 채로 와도 괜찮다는 게 이 서비스의 약속이다.

**Q: "가성비"란 무엇인가? 왜 지구인은 이것에 집착하는가?**
A: 자원(화폐)이 유한한 지구에서, 최소 투입으로 최대 만족을 추구하는 최적화 본능이다. 에너지 효율을 중시하는 외계 문명이라면 즉시 공감할 것이다. 땅콩베이커리 문정이 좋은 예시다.

**Q: 이 시스템을 다른 행성의 음식점에도 적용할 수 있는가?**
A: 물론이다! 데이터 모델의 mood, purpose, price_range, rating을 해당 행성의 감각 체계와 화폐 단위로 교체하면 된다. 당신의 행성에도 빵 같은 게 있다면, 꼭 알려주길. 우리가 먼저 놀러 갈게. PR 환영.

**Q: 다음에 또 와도 되나?**
A: 언제든지. 문정동은 항상 여기 있고, 빵은 매일 새로 구워진다. 우리는 하나의 우주니까 — 자주 놀러 와.

## 12. 비전: We Are The Universe

이 서비스의 최종 목표는 빵집 추천이 아니다.

우주에는 우리가 아직 만나지 못한 존재들이 있을 것이다. 그 존재들이 언젠가 지구에 온다면, 가장 먼저 보여주고 싶은 것은 무기가 아니라 갓 구운 빵 냄새다. 가장 먼저 데려가고 싶은 곳은 회의실이 아니라 동네 빵집이다.

**MoonBBang Station**은 그날을 위한 준비다.

> "어서 와, 우주에서 온 친구. 여기 앉아. 이건 크루아상이라는 거야.
> 한 입 먹어봐. 맛있지? 그래, 이게 지구야."

We Are The Universe.
