MoonBBang Station

우주에서 온 친구들을 위한 문정동 베이커리 가이드 시스템. 최적의 빵집 TOP 5를 추천하는 아름다운 정거장의 설계도를 소개합니다.

01. Core Technologies

🐍 Python 3.11

안정적인 백엔드 코어

풍부한 라이브러리 생태계와 직관적인 문법으로 데이터 파이프라인과 분석 로직을 신속하게 구축하기 위해 선택했습니다.
⚡ FastAPI

고성능 비동기 웹 프레임워크

비동기 처리를 통한 빠른 응답 속도와 자동화된 스웨거 문서화로 프론트엔드-백엔드 협업 효율을 극대화했습니다.
🎭 Playwright

데이터 정밀 스크래핑

최신 웹 브라우저 엔진을 완벽하게 제어하여 카카오맵, 네이버 등의 동적 콘텐츠를 정확하게 수집하기 위해 도입했습니다.
🎨 Jinja2

서버 사이드 템플릿 렌더링

FastAPI와의 높은 호환성 및 강력한 제어문을 통해 동적인 추천 결과 페이지를 유연하게 렌더링하기 위해 사용했습니다.
🧪 Pytest

신뢰성 높은 TDD 환경

복잡한 추천 로직과 데이터 검증 과정을 자동화된 테스트로 관리하여 코드 품질과 안정성을 보장하기 위해 도입했습니다.
🗺️ Kakao Maps

위치 정보 시각화 SDK

국내 위치 데이터에 특화된 지도 API를 통해 사용자에게 가장 정확하고 시각적인 빵집 위치 정보를 제공하기 위해 선택했습니다.

02. Architecture Modules

📦 Data Pipeline

scripts/ & app/data.py
카카오 로컬 API, 네이버 검색, 서울 공공데이터를 수집·정제하고 인메모리에 로드하는 데이터 공급망입니다.

🧠 Core Engine

app/recommender.py
사용자의 필터와 감각 정보를 바탕으로 가용 베이커리에 점수를 부여하고 TOP 5를 선별하는 핵심 로직입니다.

🔍 Analysis & Logic

review_analyzer.py & sensory.py
리뷰 텍스트에서 특징 태그를 추출하고, 모호한 감각 언어를 시스템 필터로 변환하는 지능형 모듈입니다.

🛸 Web Interface

app/main.py & templates/
FastAPI 라우팅과 Jinja2 템플릿, 그리고 우주 테마의 디자인 시스템이 결합된 사용자 접점입니다.

04. Engineering Highlights

🛡️ Data Integrity

3단계 검증 필터(업태·상호·양성 키워드)와 중복 제거 로직을 통해 데이터 무결성을 유지합니다. 인허가 데이터와 실시간 맵 데이터를 교차 검증하여 허위 정보를 원천 차단합니다.

📝 Living Documentation

Tech Spec, 개발 페이즈별 Plan, Walkthrough 문서를 실력 있는 엔지니어링 팀 수준으로 관리합니다. 코드와 문서의 동기화를 통해 프로젝트의 지속 가능성을 보장합니다.

📐 Clean Architecture

관심사 분리(SoC) 원칙에 따라 데이터 수집, 추천 엔진, 웹 인터페이스 모듈을 독립적으로 구성하여 유지보수성과 확장성을 극대화했습니다.

06. Development Journey

Phase 1-6

코어 엔진 및 서비스 골격

데이터 모델 설계, 리뷰 분석 알고리즘 구축 및 FastAPI 기반의 핵심 추천 API와 웹 골격을 완성했습니다.

Phase 7-10

경험의 확장: 우주적 연대

외계 친화적 온보딩, 감각 기반 추천(식감/맛) 경로를 추가하여 어느 행성에서 왔든 쉽게 빵집을 찾을 수 있게 했습니다.

Phase 11-14

데이터 생태계 및 지도 연동

카카오/네이버 시스템 연동 및 공공데이터를 도입하고, 카카오맵 JS SDK를 통한 위치 시각화 기반을 마련했습니다.

Phase 15-18

프리미엄 디자인 시스템

앰버/크러스트 디자인 시스템과 CSS Grid, 4px Spacing 체계를 도입하여 우주의 미학을 담은 UX를 완성했습니다.

Phase 19

데이터 품질 고도화

3단계 필터링으로 검색 신뢰도를 높이고, 네이버 스크래핑을 통합하여 더 풍부한 빵집 정보를 제공하게 되었습니다.
